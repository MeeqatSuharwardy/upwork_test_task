   npm run nango:start

   open http://localhost:3003/admin
   # Login: admin@example.com / admin123
   # Get your API keys from Settings

3. Configure Environment:
   cp .env.example .env
   # Add your Nango keys + Stripe credentials

4. Run the Application:
   npm run dev

5. Open Browser:
   http://localhost:3000

6. Connect Stripe:
   Click "Connect Stripe" and authorize!
   Watch data sync automatically! 🆕
